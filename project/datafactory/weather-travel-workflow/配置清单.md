### 脚本 ②：天气出游分析（Python）

基本信息：编码 `PY_WEATHER_ANALYZE` / 名称 `天气出游分析脚本` / 类型 PYTHON / 描述 解析天气JSON综合评分

DEV 版本：超时 `60` / 解释器 `/usr/bin/python3` / 最大内存 `512` / 变更说明 初始化

```python
import json

class Script:
    def execute(self, weather_raw=""):
        """gRPC 调用: kwargs 与输入参数名一致"""
        data = json.loads(weather_raw) if isinstance(weather_raw, str) else weather_raw
        # 上游可能是 API 返回的 {"statusCode":200, "body":"..."} 包装格式，解包
        if isinstance(data, dict) and "body" in data and "statusCode" in data:
            w = json.loads(data["body"]) if isinstance(data["body"], str) else data["body"]
        else:
            w = data
        c = w.get("current_condition", [{}])[0]
        temp   = int(c.get("temp_C", 0))
        desc   = c.get("weatherDesc", [{}])[0].get("value", "未知")
        wind   = int(c.get("windspeedKmph", 0))
        hum    = int(c.get("humidity", 0))
        precip = float(c.get("precipMM", 0))
        uv     = int(c.get("uvIndex", 0))

        score = 100; warns = []
        dl = desc.lower()
        if any(x in dl for x in ["rain","drizzle","shower","雨"]):    score-=30; warns.append("有降水")
        if any(x in dl for x in ["snow","sleet","hail","雪","冰雹"]): score-=40; warns.append("有降雪/冰雹")
        if any(x in dl for x in ["fog","mist","霾","雾"]):            score-=15; warns.append("能见度低")
        if   temp < 5:  score-=25; warns.append("温度过低")
        elif temp < 12: score-=10; warns.append("温度偏凉")
        elif temp > 37: score-=30; warns.append("温度过高")
        elif temp > 32: score-=10; warns.append("温度偏高")
        if   wind > 40: score-=20; warns.append("风力过大")
        elif wind > 25: score-=10; warns.append("风力较大")
        if precip > 5: score-=20; warns.append("降水量大")
        if uv > 7:     score-=10; warns.append("紫外线强")

        score = max(0, score)
        if   score >= 80: sug = "非常适合出游，祝旅途愉快！"
        elif score >= 60: sug = "适合出游，建议做好防护措施"
        else:             sug = "不太适合出游，建议改期或选择室内活动"

        return {
            "城市": w.get("request",[{}])[0].get("query","未知"),
            "温度_摄氏度": temp, "体感温度_摄氏度": int(c.get("FeelsLikeC",temp)),
            "天气状况": desc, "风速_kmh": wind, "湿度_百分比": hum,
            "降水量_mm": precip, "紫外线指数": uv, "综合评分": score,
            "是否适合出游": "是" if score>=60 else "否",
            "出游建议": sug,
            "风险提示": "；".join(warns) if warns else "无明显风险"
        }
```

> **引擎执行流程**：gRPC 客户端把 `weather_raw` 的值作为 kwargs 传给 `Script().execute(weather_raw="...")`。返回 dict 自动 JSON 序列化。**不需要** `main`、`stdin`、`sys.argv`。依赖 Python gRPC 服务在 50051 端口运行（启动脚本 4 号）。

### 脚本 ③：出行通知（Shell → Windows CMD）

基本信息：编码 `SH_TRAVEL_NOTIFY` / 名称 `出行通知脚本` / 类型 SHELL / 描述 解析上游结果输出出行通知
